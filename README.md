# Tienda Online

Proyecto de **tienda online funcional** desarrollado con **HTML, CSS, JavaScript y PHP**, enfocado en demostrar fundamentos sólidos de desarrollo web, estructuración de proyectos y lógica de e-commerce.

Este repositorio forma parte de mi **portfolio profesional**.

Este repositorio contiene el frontend completo y la lógica backend básica, **excluyendo deliberadamente** los archivos sensibles relacionados con la pasarela de pago.

---

##  Características

- Diseño responsive adaptable a móviles y escritorio  
- Catálogo de productos  
- Interacción dinámica con JavaScript  
- Backend en PHP  
- Estructura clara y mantenible  
- Enfoque en buenas prácticas

---
### Frontend
- Catálogo de productos
- Detalle de producto
- Carrito de compras
- Interacción dinámica con JavaScript y AJAX
- Diseño responsive

### Panel de administración
- Login de administrador
- CRUD de productos
- Dashboard con gráficas
- Gestión de ventas
- Paginación y manejo de datos

---

## Descripción del proyecto

La aplicación incluye un **frontend de tienda**, un **panel de administración** y lógica backend desarrollada en PHP, sin uso de frameworks, con el objetivo de reforzar y demostrar conocimientos base del desarrollo web.

Por razones de seguridad, los archivos relacionados con pasarela de pago no se incluyen en el repositorio.

---
##  Tecnologías utilizadas

- **HTML5** — estructura semántica
- **CSS3** — estilos y diseño responsive
- **JavaScript (Vanilla)** — interactividad y AJAX
- **PHP** — lógica del servidor
- **MySQL** — base de datos
